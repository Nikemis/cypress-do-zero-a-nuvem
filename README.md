ℹ️ **Sobre este repositório**

 Este repositório foi criado para acompanhar meus estudos no curso **“Cypress, do Zero à Nuvem ☁️”**.

 O projeto é um **fork do repositório original do curso**, e aqui registro toda a minha evolução com testes automatizados.

 Ao longo do aprendizado, estou:
 - Reproduzindo os exemplos apresentados no curso
 - Criando novos testes além dos propostos, para aprofundar os conhecimentos
 - Explorando diferentes abordagens de teste e boas práticas
 - Documentando o que aprendo, para servir tanto como material de consulta quanto para demonstrar minhas habilidades

 O objetivo deste repositório é mostrar minha evolução contínua como QA e automatizadora de testes, servindo como portfólio público do meu desenvolvimento.  


Sinta-se à vontade para acompanhar os commits e a construção dos testes — feedbacks são sempre bem-vindos! ✨
